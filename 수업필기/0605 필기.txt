 일반적으로 한번에 뽑는 개수가 30개 이상일때. 횟수는 무한히
  cf) 대수의 법칙: 모평균과 유사하게 된다

  2. 표준편차를 모를 때
    2-1. 등분산 t검정
    2-2. 이분산 t검정
-두 집단이 독립이 아닌경우: 상대 t검정

<다시>
1.중심극한 정리
:표본의 크기가 크면(n>=30) 
 표본들의 분포가 정규분포를 이룬다.
2. 대수의 법칙
:표본이 크면, 
표본 평균분포의 평균이 모평균에 수렴한다.
표본의 표준편차가 모표준편차/루트N 에 수렴한다.
3.표본은 
  X' ~ N(뮤, 모표준편차/루트N)
  뮤 - 1.96*(모표준편차/루트N) <= X' <= 뮤 + 1.96*(모표준편차/루트N)
따라서,  모평균의 신뢰구간 구하기
  X' - 1.96*(모표준편차/루트N) <= 뮤 <= X' + 1.96*(모표준편차/루트N)

두 집단의 "표본의 평균"으로 비교한다.(독립일때) 
1. 모집단의 세타1과 세타2가 주어졌을 때,
  X1' X2' 가 정규분포를 따른다면,  N( M1-M2  루트(세타1제곱/N1 + 세타2제곱/N2) )
2. 두집단의 분산을 모를 때
  2-1. 분산이 같을 때 = 등분산 t검정
  2-2. 분산이 다를 때 = 이분산 t검정
  2-3. 같은지 다른지 모를 때 
         F검정 - S1과 S2가 같을 때 = 등분산 t검정
                 - S1과 S2가 다를 때 = 이분산 t검정
(독립이 아닐 때, 연관일때) 상대 t검정

두 학생의 점수차이는 크지않다.

<9장 분산분석>
(그룹 내) 개별 데이터간의 편차: SSE(또는 SSW)
그룹간의 편차: SSB
SSE + SSB = SST :총편차

SSE/N = MSE
SSB/N = MSB

MSB/MSE = F비: 그룹끼리의 편차가 그룹내 편차보다 큰지
